## Codebase Patterns
- Feature flags use `z.enum(["true", "false"]).optional().default("false")` pattern in `env.mjs` with matching `runtimeEnv` entry
- app.mento.org has no `check-types` script; run `tsc --noEmit` from its directory or `pnpm check-types` from root (covers packages only)
- SDK version `^3.0.0-beta.18`; managed via pnpm workspace catalog in `pnpm-workspace.yaml`
- Features barrel is at `packages/web3/src/features/index.ts` — add new features alphabetically
- Commitlint rejects subjects starting with uppercase IDs like "US-006 - Create..." — move ID to end: `feat: add foo (US-006)`
- `LoanDetails` has no `collateralRatio` field — compute as `10n ** 36n / ltv` (inverted LTV, 18-decimal result)
- `BorrowService` constructor: `new BorrowService(publicClient: PublicClient, chainId: number)`
- `useBorrowService()` returns `BorrowService | null` — always guard with `enabled: !!sdk` in useQuery hooks
- Borrow hooks barrel at `packages/web3/src/features/borrow/hooks/index.ts` — exports alphabetically
- Borrow component directory: `apps/app.mento.org/app/components/borrow/` — dashboard/, shared/, atoms/
- `borrow-view.tsx` is the top-level router for the borrow tab — reads `borrowViewAtom` and renders sub-views
- `BorrowView` union discrimination: use `typeof view === "object"` for the `manage-trove` variant, string comparison for the rest
- Borrow navigation atom at `components/borrow/atoms/borrow-navigation.ts` — uses `BorrowView` type from `@repo/web3`
- SDK `CallParams` strings must be cast: `to` → `Address`, `data` → `Hex`, `value` → `BigInt(value || 0)`
- SDK pure math functions: import from `@mento-protocol/mento-sdk/dist/services/borrow/borrowMath`
- SDK borrow helpers (e.g., `resolveAddressesFromRegistry`): import from `@mento-protocol/mento-sdk/dist/services/borrow/borrowHelpers`
- `getBorrowRegistry(chainId, symbol)` is exported from top-level `@mento-protocol/mento-sdk`
- Formatting utils in `features/borrow/format.ts`: `formatDebtAmount`, `formatCollateralAmount`, `formatPrice`, `formatInterestRate`, `formatLtv` — all accept `bigint | null | undefined`
- `DEBT_TOKEN_CONFIGS` in `features/borrow/types.ts` — typed as `Record<string, DebtTokenConfig>` so use `!` assertion when accessing by key
- Bigint values in React Query keys must be `.toString()` converted — bigints aren't JSON-serializable
- Shared `useDebounce` utility at `packages/web3/src/utils/debounce.ts` — reuse it, don't create per-feature copies
- @repo/ui `Select` pattern: `Select` > `SelectTrigger` > `SelectValue` + `SelectContent` > `SelectItem` (Radix-based)
- @repo/ui `Badge` variants: `default`, `secondary`, `destructive`, `outline`
- @repo/ui `ChartContainer` exported from `@repo/ui` — wraps Recharts' `ResponsiveContainer` with theme CSS variables; use `var(--color-<key>)` for fills
- @repo/ui Card composable pattern: `Card` > `CardHeader` > `CardTitle` + `CardAction`, `CardContent` for body
- @repo/ui `Tabs` pattern: `Tabs` > `TabsList` > `TabsTrigger` + `TabsContent` (Radix-based)
- @repo/ui `Slider` component: Radix-based, props: `min`, `max`, `step`, `value` (array), `onValueChange`
- @repo/ui `Button` variants: `default`, `outline`, `destructive`, `ghost`, `link`
- @repo/ui `Dialog` components: `Dialog`, `DialogContent`, `DialogHeader`, `DialogTitle`, `DialogDescription`, `DialogFooter` — Radix-based, use `open` prop for programmatic control
- @repo/ui Chart (Recharts 3.6.0 wrapper): `ChartContainer` + standard Recharts components, theme-aware colors
- SDK borrow types: `BorrowPosition`, `LoanDetails`, `SystemParams`, `CallParams`, `OpenTroveParams`, `AdjustTroveParams`, `InterestRateBracket`, `TroveStatus`, `RiskLevel`
- Derived hooks use `useMemo` (not useQuery) — they compute from existing query data, not fetch new data
- `borrowViewAtom` lives in app layer (`components/borrow/atoms/borrow-navigation.ts`), NOT in packages/web3 — navigation handled by app components
- `selectedDebtTokenAtom` in `packages/web3/src/features/borrow/atoms/deployment-atoms.ts` — provides current symbol (e.g., "GBPm")
- `borrowFlowAtom` in `packages/web3/src/features/borrow/atoms/flow-atoms.ts` — persisted to localStorage, auto-shows FlowDialog when not null
- `FlowDialog` is self-managing — just render `<FlowDialog />` in borrow-view, no props needed
- App components use `@repo/web3` dist — rebuild web3 package (`pnpm --filter @repo/web3 build`) after adding new exports before running app tsc
- Block explorer tx link pattern: `${useExplorerUrl()}/tx/${txHash}` — `useExplorerUrl` from `@repo/web3`
- Write hook pattern: `useMutation` + `executeFlow(wagmiConfig, setFlowAtom, flowId, operation, account, stepDefs)` + `onSuccess` invalidates + toast
- `wagmiConfig` obtained via `useConfig()` from `wagmi`; `account`/`address` from `useAccount()` via `@repo/web3/wagmi`
- Bigint conversion: `parseUnits(inputString, 18)` for user input → bigint, `formatUnits(bigintValue, 18)` for display (from viem)
- Position card click navigates to `{ view: 'manage-trove', troveId }` via borrowViewAtom
- Dashboard components in `dashboard/` — `borrow-dashboard.tsx`, `position-card.tsx`, `stability-card.tsx`
- Shared components in `shared/` — `debt-token-selector.tsx`, `flow-dialog.tsx`, `flow-step.tsx`, `risk-badge.tsx`, `trove-metrics.tsx`
- Available read hooks: `useSystemParams`, `useCollateralPrice`, `useLoanDetails`, `useDebtSuggestions`, `useInterestRateChartData`, `useRedemptionRisk`, `usePredictUpfrontFee`, `useNextOwnerIndex`, `useTroveData`, `useUserTroves`, `useBorrowAllowance`, `useBranchStats`, `useInterestRateBrackets`
- Available write hooks: `useOpenTrove`, `useAdjustTrove`, `useCloseTrove`, `useAdjustInterestRate`, `useClaimCollateral`, `useSpDeposit`, `useSpWithdraw`
- `useOpenTrove` accepts `{ symbol, params: OpenTroveParams, wagmiConfig, account }` — OpenTroveParams: `{ owner, ownerIndex, collAmount, boldAmount, annualInterestRate, maxUpfrontFee }`
- `useAdjustTrove` accepts `{ symbol, params: AdjustTroveParams, wagmiConfig, account }` — AdjustTroveParams: `{ troveId, collChange, isCollIncrease, debtChange, isDebtIncrease, maxUpfrontFee }`
- `useCloseTrove` accepts `{ symbol, troveId, wagmiConfig, account }`
- `useAdjustInterestRate` accepts `{ symbol, troveId, newRate, maxUpfrontFee, wagmiConfig, account }`
- `useClaimCollateral` accepts `{ symbol, wagmiConfig, account }`
- Reference form pattern: `apps/app.mento.org/app/components/pools/add-liquidity-form.tsx` — multi-step approval + submit, dynamic calculations, button state machine
- Collateral token address: `getTokenAddress(chainId, "USDm" as TokenSymbol)` from `@mento-protocol/mento-sdk` — requires `TokenSymbol` cast
- ERC-20 balance pattern: `useReadContract({ address, abi: erc20Abi, functionName: "balanceOf", args: [address] })` — returns raw `bigint`
- App-layer typecheck: `cd apps/app.mento.org && npx tsc --noEmit` (not covered by `pnpm check-types`)

  - Manual input uses `inputMode="decimal"` for mobile numeric keyboard, with regex validation for positive decimals only
